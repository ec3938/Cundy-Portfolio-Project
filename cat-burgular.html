The Cat Burgular

Game Programming Final Project

Description

This is our final project for NYU's Intro to Game Programming class. This game is an endless runner, drawing inspiration from games like Subway Surfers, Jetpack Joyride, and the Dinosaur game from Google. Try to get away from Dog Cop for as long as possible, but beware the speed gets faster and faster the longer you run. Pick up power-ups along the way to get increase your score and help you get away and avoid the pigeons that slow you down and the trash cans that'll stop you dead in your tracks. All assets in this game except the SFX are made by us. To jump simply tap the screen and if you swipe downwards on the screen you'll fall straight to the ground.
